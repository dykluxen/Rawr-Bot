# Rawr-Bot
Discord bot for fun interactions and community ideas
This is a Discord bot I built to make servers feel warmer and more fun. It’s centered around cute interactions and friendly commands that give people a simple way to express themselves and connect with others.

There are many interaction commands to choose from, such as boops, hugs, pats, nuzzles, glomps, cuddles, purrs, and more. Each one has its own custom message, so it doesn’t feel generic. You can send a rawr, kick off a boop battle, or just make someone’s day a little better with a tail wag.

It also has a bunch of extra features that are just fun to mess around with. Things like random cute noises, vibe checks, cuteness ratings, fortune messages, sparkles, snacks, gifts, and a cutieball that gives sweet little answers to your questions. Nothing too serious, just stuff that makes the server feel alive.

If you need actual server info, there are basic utility commands too, like /whois, /serverinfo, and /ping. They fit the same style as everything else, so the bot stays consistent.

Overall, this bot is something I made because I wanted servers to feel more personal and fun. It helps people interact in ways that go beyond just chatting, and it brings a cute, lighthearted vibe to whatever kind of server you run. :3
